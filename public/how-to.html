API: Endpoints

  • Current endpoints include:
    • api
    • api/titles
    • api/date or api/dates
      • api/date/month
      • api/date/month/day
      • api/date/month/day/year
    • api/dedicateds
    • api/database
    • api/sheetinfo
    • csvinfo
    • csvinfo/titles
    • csvinfo/date or api/dates
      • csvinfo/date/month
      • csvinfo/date/month/day
      • csvinfo/date/month/day/year
    • refresh
    • refresh/id
    • checkzips
    • zipnames
    • zipnames/id
    • zipname
    • zipname/id
    • csvnames
    • csvnames/id
    • csv
    • function/id (internal)
    • trello/id (internal)
  • api
    • Returns all of the cells requested
  • api/titles
    • Returns the first row from the googlesheet, (usually the titles)
  • api/date or api/dates
    • Return dates of all the dayparts with no duplicates
  • api/date/[month]
    • Return the dayparts with the associated month, no duplicates
  • api/date/[month]/[day]
    • Return the dayparts with the associated month and day, no duplicates
  • api/date/[month]/[day]/[year]
    • Return the dayparts with the associated month, day and year, no duplicates
  • csvinfo
    • Returns all of the cells requested
  • csvinfo/titles
    • Returns the first row from the googlesheet, (usually the titles)
  • csvinfo/date or api/dates
    • Return dates of all the dayparts with no duplicates
  • csvinfo/date/[month]
    • Return the dayparts with the associated month, no duplicates
  • csvinfo/date/[month]/[day]
    • Return the dayparts with the associated month and day, no duplicates
  • csvinfo/date/[month]/[day]/[year]
    • Return dayparts with the associated month, day and year, no duplicates
  • refresh
    • Returns new sheet data from currently loaded sheet
  • refresh/[id]
    • Returns new sheet's data
  • checkzips
    • Get list of all zips including extentions
  • zipname
    • Get list of all zip names, no extentions
  • zipname/[zipName]
    • Check if zip exists, returns empty string if doesnt exist
  • zipnames
    • Get list of all zip names, no extentions
  • zipnames/[zipName]
    • Check if zip exists, returns empty string if doesnt exist
  • csvnames
    • Get list of csvs
  • csvnames/[csvName]
    • Check if csv exists, returns empty string if doesnt exist
  • csv (internal)
    • Upload csv for reading/saving via the frontend
  • function/[functionName] (internal)
    • Run functions that return data from backend
    • Run functions that output content
  • trello/[documentName] (internal)
    • Download trello json
    • Add to trello folder
    • Use name of trello json file as documentName
    • gives list of call cards in their respective lanes

Errors

  • Unable to parse range...
    • Your tab title maybe incorrectly spelled in the database
    • some times the wrong tab from the wrong sheet is saved in the defaults
      1. Go to your sheet in google
      2. copy the name of the tab you wish to view
      3. go to settings in the app (cog in the top left)
      4. add your tab name in the tab area
      5. click submit
    • The tab in the default does not exist in the sheet
      1. Copy/paste a different tab name into the defaults
      2. go to settings in the app (cog in the top left) and paste the tab name
      3. click submit
      4. OR add a tab by the same name to your sheet
      5. OR change the name of a tab in your sheet to the name in the defaults
  • The caller does not have permission
    • The app does not have permission to read your sheet
      1. Go to your sheet in google
      2. click the share button
      3. add the email hpto-node@hpto-1617047136949.iam.gserviceaccount.com
