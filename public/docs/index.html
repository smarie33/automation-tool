HPTO Code

  • All content must fit into a 800px x 235px rectangle.
  • Background of content in iframe must be transparent on final delivered
  • Use the desktop applet Make HPTO Duplicates after your finished building to create mocks for approval. Download here
  • Download the most recent css here
  • Download the most recent js here
  • Download the most recent html here
  • How to get the app to work on your computer
    1. Download the zip
    2. Unzip anywhere on your computer to get the MakeHPTOduplicates.app
    3. Create a new folder and name it something you will remember
    4. Right-click on MakeHPTOduplicates.app and choose "Show Package Contents"
    5. Click-drag the contents of MakeHPTOduplicates.app in the new folder you created
    6. Rename the new folder with ".app" at the end of the name
    7. Click "add" when the dialog box asks if you want to add this extension
    8. Now the app should be usable on your computer,bypassing the apple gate
